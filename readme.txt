This plug-in will create a vertical scroll news or Announcement or flash News for your word press site, we can embed this in site sidebar.  	

1. Easy installation.  
2. Widgets, so you can add pretty much anything.  
3. Easy style-override system.  
4. Expiration date setup.  
5. You can add N number of news; it will scroll one by one at front end (vertical scrolling).   
6. You can arrange the scrolling news order. Also you can customize the scroll direction.  
7. If you want, you can hide the news temporarily.   
8. Only ADMIN user can access this plug-in.

<code><?php if (function_exists (news_announcement)) news_announcement(); ?></code>		

**To configure:**

*   Go to Announcement link under setting tab to add/edit/delete Announcement.
*   Go to widget link under Appearance tab, Drag & drop announcement widget to your desired location in the active sidebar. 
*   Click on the configure button (small down triangle) for the "news announcement" widget and here you can customize all the Announcement front end styles.

== Installation ==

**Installation Instruction**

*   Unpack the *.zip file and extract the /news-announcement-scroll/ folder.	
*   Drop the 'news-announcement-scroll' folder into your 'wp-content/plugins' folder	
*   In word press administration panels, click on plug-in from the menu tab.	
*   You should see your new Announcement plug-in listed under Inactive plug-in tab.	
*   To turn the word presses plug-in on, click activate.	
  
**To configure**

*   Go to Announcement link under setting tab to add/edit/delete Announcement.	
*   Go to widget link under Appearance tab, Drag & drop "news announcement" widget to your desired location in the active sidebar. 	
*   Click on the configure button (small down triangle) for the "news announcement" widget and here you can customize all the Announcement front end styles.	

== Frequently Asked Questions == 

**Can I display more news at same time?**  
Yes, you can add n number of news, it will display one by one at front end (vertical scrolling).  
  
**Can I change the scroll manner from vertical to horizontal?**  
No,  
  
**Can I display announcement in random order?**  
Yes, Option available in 6.0 version.	
	
**Can I arrange the news scroll order?**  
Yes, you can order, use display order field to arrange the news scroll order.  
  
**Can I hide the any news temporary?**  
Yes, change the display order status to OFF.  
  
**Can I set expiration for announcement?**  
Yes,Use version 5.0.

**Can I change the Slide Direction?**  
Yes, 0=down-up and 1=up-down.  
  
**Can I customize the all other appearance style?**  
Yes, Click on the configure button (small down triangle) for the "news announcement" widget and here you can customize all the Announcement front end styles. See screen two.

**Why my news content out of range?**  
In front end widget area, if you see any news content out of area or invisible, it because of height and width of the widget, so you should arrange width and height of the widget in widget configuration area to good look. In default I have fixed width: 180px and height: 100px.

**How to type other then english?**  
Simply copy your other language content and past it into announcement.
